# v0.2.0 Namespace Refactor Eval Results

Date: 2026-04-30

## Focus

v0.2.0 introduces the canonical Records plugin namespace:

```text
/records:fhir-validation
```

The old invocation remains available as a deprecated compatibility plugin:

```text
/records-fhir-validation:records-fhir-validation
```

## Checks

| Check | Result | Notes |
|---|---|---|
| New plugin structure | Pass | `plugins/records` contains plugin metadata, privacy/docs/evals, and `skills/fhir-validation/SKILL.md`. |
| Marketplace entries | Pass | Marketplace includes canonical `records` plugin and deprecated `records-fhir-validation` compatibility plugin. |
| Smoke test | Pass | `npm run test:claude-plugin` validates both plugin entries and the CLI path. |

## Release Decision

v0.2.0 is suitable for publication as the canonical namespace release.
