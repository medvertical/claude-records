# Records Claude Plugin Eval Results v0.3.0

## Scope

Operational feature release:

- Add release-check script that runs local tests, Claude plugin validation, version sync, current eval-result checks, and live Claude update checks when `claude` is installed.
- Add tool mock runtime tests for Records CLI, SUSHI, Firely Terminal, and HAPI detection.
- Add executable OperationOutcome explainer.
- Add executable quality-rule derivation helper.
- Add executable GitHub Actions CI generator.
- Add FHIR expression to JSON Pointer mapper.
- Expand PHI redaction summaries for identifiers, Bundles, contained resources, references, and risk levels.
- Add compatibility matrix for Records, SUSHI, IG Publisher, Firely, HAPI, and structural fallback.
- Add golden fixture snapshots for detector, generated-to-FSH mapping, and redacted Patient summaries.
- Include dark Records signet assets in root and plugin asset folders.
