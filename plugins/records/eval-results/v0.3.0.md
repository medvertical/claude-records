# Records Claude Plugin Eval Results v0.3.0

## Scope

This release fixes the public naming model before broader launch:

- Marketplace namespace is now `medvertical`.
- Canonical install name is now `records@medvertical`.
- Skill invocation remains `/records:fhir-validation`.
- Deprecated `records-fhir-validation` compatibility plugin is no longer published in the exported marketplace bundle.

## Release Decision

Ship v0.3.0 as the clean pre-launch namespace release.
