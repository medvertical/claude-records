## Regressions Caught

1. The skill needed a stronger rule against placeholder clinical fixes.
2. The skill and README incorrectly referenced a `records validate-file --server` flag. The current CLI uses global `--api-url` and `--auth-token`.
3. The IG-folder flow needed to state the structural-vs-profile boundary even when asking for a missing path.

## Release Decision

v0.1.1 is suitable for publication as a prompt-hardening and CLI-correctness release.
